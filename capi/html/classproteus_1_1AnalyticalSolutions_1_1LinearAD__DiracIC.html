Detailed Description

The exact solution of

.. math:: u_t + \nabla \cdot (b u - a \nabla u) = 0

on an infinite domain with dirac initial data

.. math:: u0 = \int u0 \delta(x - x0)

also returns advective, diffusive, and total flux
