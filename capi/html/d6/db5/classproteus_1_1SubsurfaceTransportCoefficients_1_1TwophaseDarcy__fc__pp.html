Detailed Description

continuity equation for each phase

.. math::

  \pd{m_w}{t} - \deld (\ten{a}_w \grad \phi_w) + r_w = 0
  \pd{m_n}{t} - \deld (\ten{a}_n \grad \phi_n) + r_n = 0
